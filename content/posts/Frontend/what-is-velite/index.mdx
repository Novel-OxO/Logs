# Velite란 무엇인가요?

**Velite**는 로컬 파일(Markdown, MDX, JSON 등)을 정적 데이터로 변환해주는 강력한 도구입니다. 특히 Next.js와 같은 최신 프레임워크에서 블로그나 문서를 구축할 때 매우 유용합니다.

## 왜 Velite인가요?

기존에는 Markdown 파일을 읽기 위해 `fs` 모듈을 사용하고 `gray-matter`로 프론트매터를 파싱하는 등 번거로운 과정이 필요했습니다. 하지만 Velite를 사용하면 다음과 같은 장점을 얻을 수 있습니다:

1. **안전한 데이터**: 스키마를 정의하므로 필수 필드가 누락되는 일을 방지할 수 있습니다.
2. **개발자 경험**: TypeScript 자동 완성을 통해 데이터 구조를 즉시 파악할 수 있습니다.
3. **확장성**: `remark`, `rehype` 플러그인을 쉽게 연동하여 코드 하이라이팅이나 수식 입력 등을 구현할 수 있습니다.

## 시작하기

이미 프로젝트에 Velite 설정이 완료되어 있습니다. `velite.config.ts`에서 스키마를 확인해보세요!

```typescript
// velite.config.ts 예시
posts: {
  name: 'Post',
  pattern: 'posts/**/*.mdx',
  schema: s.object({
    title: s.string(),
    date: s.isodate(),
    // ... 더 많은 필드
  })
}
```

이제 `content/posts` 폴더에 `.mdx` 파일을 추가하기만 하면 자동으로 블로그 데이터로 등록됩니다.
